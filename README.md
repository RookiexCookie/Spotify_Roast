1. You sign in with Spotify (via NextAuth). The server fetches your top tracks/artists (or accepts a playlist URL).
2. The app sends the items to `POST /api/roast`, where a strict AI prompt (see `src/app/api/roast/route.ts`) crafts:

   - A numeric score (0â€“100)
   - A roast for each item (1â€“2 sentences, Hinglish-ish, savage)
